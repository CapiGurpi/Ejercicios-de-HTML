RESPOSTES

Respon a les següents preguntes en un document HTML de nom “respostes.html”

Sobre el display INLINE-BLOCK

  • Com podem situar una caixa al costat de l’altre?
    • Utilitzant el format "display: inline-block", això fa que es presentin tots en línea i permet posar un element al costat de'l altre.
  • Què passa si una caixa no hi cap al costat?
    • Que com no hi ha espai suficient la caixa passa a la següent línea.
  • Perquè serveix la propietat “vertical-align”?
    • Per alinear els elements del inline-block.
  • Com podem centrar una caixa dins d’una altre?
    • Amb el format "text-align: center" utilizat dins d'un inline-block.

Sobre el display FLEX-BOX

  • Com podem crear files?
    • Utilitzant el format "display: flex".
  • Perquè serveix flex-wrap?
    • Per indicar que elements flexibes poden canviar de línea sino caben.
  • Perquè serveixen les propietats flex-grow, flex-shrink i flex-basis?
    • Les tres serveixen per modificar la mida de l'objecte.

Sobre el display GRID-LAYOUT

  • Com indiquem el número de files i columnes de la grid?
    • Amb els formats grid-template-rows i grid-template-columns amb els que indiquem la mida i el nombre de files i columnes.
  • Com indiquem quantes files i/o columnes ocupa una àrea de la grid?
    • Amb les propietats grid-row i grid-column, que defineixen l’inici i el final de l’àrea que ocupa un element dins la graella.
  • Què indiquen colom-gap i el row-gap?
    • El primer l'espai entre columnes i el segon l'espai entre les files.
  • Què indica la propietat grid-area?
    • Es una forma d'indicar l'àrea que ocuparà un element.
